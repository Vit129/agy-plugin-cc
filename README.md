# cc-agy-plugin

Antigravity (agy) integration for Claude Code.

## Installation in Claude Code

Run this command inside Claude Code to install the plugin locally:

```text
/plugin install ./cc-agy-plugin
```

Then reload plugins:

```text
/reload-plugins
```

## Usage

You can now delegate tasks from Claude Code to Antigravity using:

```text
/cc-agy-plugin:agy <your prompt here>
```
